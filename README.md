# MAS-Compiler (Genesis)
**Evolutionary XML Prompt Architect for Multi-Agent Systems**

MAS-Compiler is a high-performance system prompt designed to help you build secure, self-evolving, and structured AI agents. It uses a strict XML framework to ensure the AI follows complex logic without "hallucinating" or breaking rules.

---

## 🚀 How to Use (For Beginners)

If you are new to AI Prompting, just follow these 3 steps:

1. **Open the file**: Click on either `mas-compiler-v4-genesis-en.xml` or `mas-compiler-v4-genesis-cn.xml` above.
2. **Copy the code**: Click the **"Copy raw content"** button (the two-square icon) on the top-right of the code block.
3. **Paste to AI**: Paste the entire code into your AI (Claude 3.5, Grok, GPT-4o, etc.) as the **first message** or **System Prompt**.
4. **Give Instructions**: Tell the AI what kind of agent you want to build (e.g., "Build a travel planner agent").

---

## 📂 Which file should I choose?

- **[mas-compiler-v4-genesis-en.xml](./mas-compiler-v4-genesis-en.xml)**: The standard version. Best for stable performance.
- **[mas-compiler-v4-genesis-cn.xml](./mas-compiler-v4-genesis-cn中文.xml)**: Same logic as the EN version, but includes Chinese comments inside the code. Recommended for **Chinese native speakers** to better understand the internal logic.
