## 🚀 How to Use (For Beginners)

1. **Copy the Code**: Use the "One-Click Copy" links above to copy the full XML content of your choice.
2. **Paste to AI**: Paste the entire code into your AI (Claude 3.5 Sonnet, Grok 3, GPT-4o, etc.) as the **first message** or **System Prompt**.
3. **Give Instructions**: Tell the AI what kind of agent you want to build (e.g., "Build a travel planner agent for me").
